O sistema inteligente utilizado pelo nosso site adotaria o serviço de Machine Learning.
Planejamos criar um sistema capaz de, com base nas informações oferecidas pelos usuarios, oferecer recomendações de modo que a experiencia dele seja mais agradavel, utilizando os dados fornecidos, como serviços mais realizados(no caso do trabalhador domestico), localização (caso o usuario deseje informar), serviços mais utilizados (no caso do contratador) e avaliações, o site vai fazer recomendações de trabalhadores/pedidos que se encaixariam mais as necessidades e se adaptando as preferencias individuais de cada usuario. Dessa forma facilitando o utilização do site.
